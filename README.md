# GridMapConverter-ros

## Installation
1. Install Grid Map for ROS Melodic
```
apt-get install ros-melodic-grid-map
```
2. Clone source code
```
cd ~/catkin_ws/src
git clone https://github.com/Hyun-je/GridMapConverter-ros/
```
3. Build package
```
cd ~/catkin_ws
catkin_make
```
